Function Reference: list_packages

pkg-octave-doc: valid_packages = list_packages ()

List pkg-installable packages from Octave Packages.

valid_packages = list_packages () returns a cell array of strings with the available names at Octave Package, which are installable with the pkg command, along with the URL to their latest release.
